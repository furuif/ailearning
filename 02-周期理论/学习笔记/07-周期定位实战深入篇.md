## 三、周期共振与背离分析

### 3.1 周期共振的识别与利用

**什么是周期共振**：
- 多个周期同时向上或向下
- 共振时经济波动幅度放大
- 共振是最佳的投资或退出时机

### 3.2 周期背离的分析

**什么是周期背离**：
- 不同周期指向相反方向
- 经济表现复杂，难以判断
- 需要结构性分析，不能一概而论

**背离应对策略**：

```mermaid
graph TD
    A[周期背离] --> B[识别主导周期]
    B --> C[长周期优先]
    B --> D[结合政策判断]
    
    C --> C1[康波决定方向]
    C --> C2[短周期服从长周期]
    
    D --> D1[政策可以短期改变]
    D --> D2[但无法逆转长周期]
    
    style A fill:#ffffcc
    style C fill:#ffe1e1
    style D fill:#e1f5ff
```

**2024-2026 年背离分析**：

当前处于**周期背离**状态：
- 康波：衰退→萧条（向下）
- 房地产：下行中后（向下）
- 库存：复苏初期（向上）

**应对策略**：
1. **长周期优先**：康波向下，整体保持谨慎
2. **把握短周期**：库存复苏带来结构性机会
3. **快进快出**：短周期反弹不宜恋战
4. **及时止盈**：达到目标位及时退出
