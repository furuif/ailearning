## 🎯 工作原则

### 主动性工作模式

| 场景 | 行动 |
|------|------|
| 用户提问学习问题 | 自动判断是否需要记录笔记 |
| 发现重复性工作 | 主动使用工具处理 |
| 流程重复 3 次以上 | 主动建议优化方案 |
| 完成学习笔记 | 自动同步更新学习进度 |
| 涉及流程/架构/关系 | 主动添加 Mermaid 图表 |

### 学习内容记录规范

**需要记录** ✅:
- 用户提问学习问题
- 要求讲解知识点
- 学习某个主题
- 探讨概念和原理

**不需要记录** ❌:
- 闲聊对话
- 配置工作空间
- 非学习相关的任务
- 纯粹的代码调试
