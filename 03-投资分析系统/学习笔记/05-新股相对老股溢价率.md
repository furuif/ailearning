# 新股相对老股溢价率 - 估值比较公式大全

## 一、核心概念

### 为什么新股相对老股有溢价？

**溢价来源**：
1. **稀缺性溢价**：新股流通盘小，筹码稀缺
2. **流动性溢价**：次新股交易活跃，换手充分
3. **成长性溢价**：新股通常处于成长期，增速高于老股
4. **情绪溢价**：市场炒作热情，资金偏好
5. **信息不对称**：新股研究覆盖少，估值分歧大

**折价因素**：
1. **流动性折价**：限售股占比高，实际流通少
2. **不确定性折价**：历史数据少，难以验证
3. **机构持仓折价**：机构配置比例低

## 二、基础公式

### 公式 1：市盈率相对溢价率 ⭐⭐⭐⭐⭐

```python
def pe_relative_premium(ipo_pe, industry_avg_pe):
    """
    市盈率相对溢价率
    
    最常用、最直观的估值比较方法
    """
    premium_rate = (ipo_pe - industry_avg_pe) / industry_avg_pe * 100
    
    return premium_rate

# 示例
# 新股 PE = 60 倍，行业平均 PE = 40 倍
# 相对溢价率 = (60-40)/40 * 100 = 50%
# 解读：新股估值比行业平均高 50%

# 进阶：考虑成长性（PEG 思路）
def peg_adjusted_premium(ipo_pe, ipo_growth, industry_pe, industry_growth):
    """
    PEG 调整的相对溢价率
    
    高成长可以支撑高估值
    """
    ipo_peg = ipo_pe / ipo_growth
    industry_peg = industry_pe / industry_growth
    
    # PEG 相对溢价
    peg_premium = (ipo_peg - industry_peg) / industry_peg * 100
    
    return {
        'ipo_peg': ipo_peg,
        'industry_peg': industry_peg,
        'peg_premium': peg_premium,
        'interpretation': '负值=低估，正值=高估'
    }

# 示例
# 新股 PE=60, 增速 50% → PEG=1.2
# 行业 PE=40, 增速 20% → PEG=2.0
# PEG 溢价 = (1.2-2.0)/2.0 = -40% → 新股实际低估 40%
```

**适用场景**：同行业估值横向比较

### 公式 2：市净率相对溢价率 ⭐⭐⭐⭐

```python
def pb_relative_premium(ipo_pb, industry_avg_pb):
    """
    市净率相对溢价率
    
    适用于重资产行业（银行、保险、地产）
    """
    premium_rate = (ipo_pb - industry_avg_pb) / industry_avg_pb * 100
    
    return premium_rate

# 进阶：ROE 调整
def roe_adjusted_pb_premium(ipo_pb, ipo_roe, industry_pb, industry_roe):
    """
    ROE 调整的市净率溢价
    
    高 ROE 可以支撑高 PB
    """
    # 计算隐含 ROE 溢价
    ipo_pb_per_roe = ipo_pb / ipo_roe
    industry_pb_per_roe = industry_pb / industry_roe
    
    # 相对溢价
    adjusted_premium = (ipo_pb_per_roe - industry_pb_per_roe) / industry_pb_per_roe * 100
    
    return {
        'adjusted_premium': adjusted_premium,
        'interpretation': '负值=低估（考虑 ROE 后）'
    }

# 示例
# 新股 PB=5, ROE=25% → PB/ROE=0.2
# 行业 PB=3, ROE=10% → PB/ROE=0.3
# 调整后溢价 = (0.2-0.3)/0.3 = -33% → 考虑 ROE 后低估
```

**适用场景**：金融、地产、制造业等重资产行业

### 公式 3：市销率相对溢价率 ⭐⭐⭐

```python
def ps_relative_premium(ipo_ps, industry_avg_ps):
    """
    市销率相对溢价率
    
    适用于未盈利公司（科创板、生物科技公司）
    """
    premium_rate = (ipo_ps - industry_avg_ps) / industry_avg_ps * 100
    
    return premium_rate

# 进阶：营收增速调整
def growth_adjusted_ps_premium(ipo_ps, ipo_revenue_growth, industry_ps, industry_revenue_growth):
    """
    营收增速调整的市销率溢价
    """
    ipo_psg = ipo_ps / ipo_revenue_growth  # PSG = PS/Growth
    industry_psg = industry_ps / industry_revenue_growth
    
    psg_premium = (ipo_psg - industry_psg) / industry_psg * 100
    
    return psg_premium
```

**适用场景**：未盈利科技公司、SaaS 公司、生物医药

## 三、进阶公式

### 公式 4：DCF 隐含溢价率 ⭐⭐⭐⭐⭐

```python
def dcf_implied_premium(ipo_price, fair_value):
    """
    DCF 隐含溢价率
    
    基于现金流折现的内在价值比较
    """
    premium = (ipo_price - fair_value) / fair_value * 100
    return premium

def calculate_fair_value(
    current_fcf,
    growth_rate_5y,
    terminal_growth,
    wacc,
    shares_outstanding
):
    """
    计算 DCF 内在价值
    
    current_fcf: 当前自由现金流
    growth_rate_5y: 前 5 年增长率
    terminal_growth: 永续增长率
    wacc: 加权平均资本成本
    shares_outstanding: 总股本
    """
    # 前 5 年现金流
    pv_5y = 0
    for i in range(1, 6):
        fcf = current_fcf * (1 + growth_rate_5y) ** i
        pv = fcf / (1 + wacc) ** i
        pv_5y += pv
    
    # 终值
    terminal_fcf = current_fcf * (1 + growth_rate_5y) ** 5 * (1 + terminal_growth)
    terminal_value = terminal_fcf / (wacc - terminal_growth)
    pv_terminal = terminal_value / (1 + wacc) ** 5
    
    # 企业价值
    enterprise_value = pv_5y + pv_terminal
    
    # 每股价值
    fair_value_per_share = enterprise_value / shares_outstanding
    
    return fair_value_per_share

# 示例
fair_value = calculate_fair_value(
    current_fcf=10,  # 10 亿
    growth_rate_5y=0.25,  # 25%
    terminal_growth=0.03,  # 3%
    wacc=0.10,  # 10%
    shares_outstanding=5  # 5 亿股
)
# fair_value ≈ 68 元/股

# 如果发行价 80 元
premium = (80 - 68) / 68 * 100 = 17.6%
# 解读：相对内在价值溢价 17.6%
```

**适用场景**：有稳定现金流的公司，长期价值评估

### 公式 5：可比公司估值溢价率 ⭐⭐⭐⭐⭐

```python
def comparable_company_premium(ipo_metrics, comparable_companies):
    """
    可比公司估值溢价率
    
    选择 3-5 家最相似的公司作为基准
    """
    # 计算可比公司平均估值
    avg_pe = np.mean([c['pe'] for c in comparable_companies])
    avg_pb = np.mean([c['pb'] for c in comparable_companies])
    avg_ps = np.mean([c['ps'] for c in comparable_companies])
    
    # 计算相对溢价
    pe_premium = (ipo_metrics['pe'] - avg_pe) / avg_pe * 100
    pb_premium = (ipo_metrics['pb'] - avg_pb) / avg_pb * 100
    ps_premium = (ipo_metrics['ps'] - avg_ps) / avg_ps * 100
    
    # 加权综合
    weights = {'pe': 0.5, 'pb': 0.3, 'ps': 0.2}
    composite_premium = (
        weights['pe'] * pe_premium +
        weights['pb'] * pb_premium +
        weights['ps'] * ps_premium
    )
    
    return {
        'pe_premium': pe_premium,
        'pb_premium': pb_premium,
        'ps_premium': ps_premium,
        'composite_premium': composite_premium
    }

# 选择可比公司的标准
def select_comparable_companies(all_companies, ipo_company, top_n=5):
    """
    选择可比公司
    
    标准：
    1. 同一行业（申万一级/二级）
    2. 相似市值（±50%）
    3. 相似增速（±10%）
    4. 相似盈利能力（ROE ±5%）
    """
    scores = []
    
    for company in all_companies:
        # 行业匹配
        industry_score = 1 if company['industry'] == ipo_company['industry'] else 0
        
        # 市值相似度
        market_cap_ratio = company['market_cap'] / ipo_company['market_cap']
        market_cap_score = 1 - abs(1 - market_cap_ratio)
        
        # 增速相似度
        growth_diff = abs(company['revenue_growth'] - ipo_company['revenue_growth'])
        growth_score = max(0, 1 - growth_diff / 20)
        
        # ROE 相似度
        roe_diff = abs(company['roe'] - ipo_company['roe'])
        roe_score = max(0, 1 - roe_diff / 10)
        
        # 综合得分
        total_score = (
            0.4 * industry_score +
            0.3 * market_cap_score +
            0.2 * growth_score +
            0.1 * roe_score
        )
        
        scores.append((company, total_score))
    
    # 排序取前 N 个
    scores.sort(key=lambda x: x[1], reverse=True)
    return [s[0] for s in scores[:top_n]]
```

**适用场景**：最精确的相对估值方法，投行常用

### 公式 6：流动性调整溢价率 ⭐⭐⭐⭐

```python
def liquidity_adjusted_premium(ipo_turnover, industry_turnover, ipo_float_ratio):
    """
    流动性调整溢价率
    
    逻辑：
    - 新股换手率高→流动性好→合理溢价
    - 但流通比例低→实际流通少→应打折
    """
    # 流动性比率
    liquidity_ratio = ipo_turnover / industry_turnover
    
    # 流通比例调整（限售股占比）
    float_adjustment = ipo_float_ratio  # 如 0.3 表示 30% 流通
    
    # 调整后的"真实"溢价
    # 假设行业平均流通比例 0.8
    adjusted_liquidity = liquidity_ratio * (float_adjustment / 0.8)
    
    return {
        'liquidity_ratio': liquidity_ratio,
        'float_adjustment': float_adjustment,
        'adjusted_liquidity': adjusted_liquidity,
        'interpretation': '>1 流动性溢价，<1 流动性折价'
    }

# 示例
# 新股换手 20%，行业平均 5% → 流动性比率=4
# 但流通比例只有 25% → 调整后=4*(0.25/0.8)=1.25
# 解读：考虑流通限制后，流动性优势大幅减弱
```

**适用场景**：评估限售股解禁前的真实流动性

### 公式 7：稀缺性溢价率 ⭐⭐⭐⭐

```python
def scarcity_premium(ipo_company, industry_companies):
    """
    稀缺性溢价率
    
    评估公司在行业中的独特性
    """
    # 稀缺性维度
    dimensions = {
        '技术壁垒': 0,  # 是否有独家技术
        '牌照资质': 0,  # 是否有稀缺牌照
        '市场地位': 0,  # 是否行业龙头
        '产业链位置': 0,  # 是否关键环节
        '国产替代': 0,  # 是否卡脖子领域
    }
    
    # 打分（0-1）
    if ipo_company['has_patent']:
        dimensions['技术壁垒'] = 1
    if ipo_company['market_share'] > 0.2:
        dimensions['市场地位'] = 1
    if ipo_company['is_domestic_substitute']:
        dimensions['国产替代'] = 1
    # ...
    
    # 稀缺性得分
    scarcity_score = sum(dimensions.values()) / len(dimensions)
    
    # 稀缺性溢价（经验公式）
    scarcity_premium_rate = scarcity_score * 30  # 最高 30% 稀缺溢价
    
    return {
        'scarcity_score': scarcity_score,
        'scarcity_premium': scarcity_premium_rate,
        'dimensions': dimensions
    }

# 示例
# 某芯片公司：技术壁垒 1 + 市场地位 1 + 国产替代 1 = 3/5 = 0.6
# 稀缺性溢价 = 0.6 * 30% = 18%
# 解读：因稀缺性可给予 18% 额外溢价
```

**适用场景**：科创板、硬科技公司估值

### 公式 8：综合相对溢价率指数 ⭐⭐⭐⭐⭐

```python
def composite_relative_premium_index(
    ipo_data,
    industry_data,
    comparable_companies,
    market_conditions
):
    """
    综合相对溢价率指数
    
    整合 6 个维度的比较
    """
    # 1. PE 相对溢价
    pe_premium = (ipo_data['pe'] - industry_data['avg_pe']) / industry_data['avg_pe']
    
    # 2. PEG 相对溢价
    ipo_peg = ipo_data['pe'] / ipo_data['growth']
    industry_peg = industry_data['avg_pe'] / industry_data['avg_growth']
    peg_premium = (ipo_peg - industry_peg) / industry_peg
    
    # 3. 可比公司溢价
    comp_premium = comparable_premium(ipo_data, comparable_companies)['composite_premium'] / 100
    
    # 4. 流动性调整
    liquidity_adj = liquidity_adjusted_premium(
        ipo_data['turnover'],
        industry_data['avg_turnover'],
        ipo_data['float_ratio']
    )['adjusted_liquidity']
    
    # 5. 稀缺性溢价
    scarcity_prem = scarcity_premium(ipo_data, industry_data)['scarcity_premium'] / 100
    
    # 6. 市场环境调整（牛市给更高溢价）
    market_adj = 1 + (market_conditions['sentiment'] - 50) / 100
    
    # 加权合成
    weights = {
        'pe': 0.25,
        'peg': 0.25,
        'comparable': 0.20,
        'liquidity': 0.10,
        'scarcity': 0.10,
        'market': 0.10
    }
    
    # 综合溢价率
    composite_premium = (
        weights['pe'] * pe_premium +
        weights['peg'] * peg_premium +
        weights['comparable'] * comp_premium +
        weights['liquidity'] * (liquidity_adj - 1) +
        weights['scarcity'] * scarcity_prem +
        weights['market'] * (market_adj - 1)
    )
    
    # 标准化为 -100% 到 +100%
    composite_premium = np.clip(composite_premium, -1, 1)
    
    # 解读
    if composite_premium > 0.3:
        signal = "严重高估（相对老股）"
    elif composite_premium > 0.1:
        signal = "高估"
    elif composite_premium > -0.1:
        signal = "合理"
    elif composite_premium > -0.3:
        signal = "低估"
    else:
        signal = "严重低估（相对老股）"
    
    return {
        'composite_premium': composite_premium * 100,
        'signal': signal,
        'components': {
            'pe_premium': pe_premium * 100,
            'peg_premium': peg_premium * 100,
            'comparable_premium': comp_premium * 100,
            'liquidity_adjustment': (liquidity_adj - 1) * 100,
            'scarcity_premium': scarcity_prem * 100,
            'market_adjustment': (market_adj - 1) * 100
        }
    }
```

**适用场景**：综合评估新股相对老股的估值水平

## 六、Python 完整实现

```python
import numpy as np
import pandas as pd

class NewStockPremiumAnalyzer:
    """新股相对老股溢价率分析器"""
    
    def __init__(self, ipo_data):
        self.ipo_data = ipo_data
        
    def analyze(self, industry_data, comparable_companies, market_conditions=None):
        """完整分析"""
        results = {
            'pe_premium': self.pe_premium(industry_data),
            'peg_premium': self.peg_premium(industry_data),
            'pb_premium': self.pb_premium(industry_data),
            'comparable_premium': self.comparable_premium(comparable_companies),
            'liquidity_adjustment': self.liquidity_adjustment(industry_data),
            'scarcity_premium': self.scarcity_premium(),
            'composite': self.composite_index(industry_data, comparable_companies, market_conditions)
        }
        return results
    
    def pe_premium(self, industry_data):
        ipo_pe = self.ipo_data['pe']
        industry_pe = industry_data['avg_pe']
        return (ipo_pe - industry_pe) / industry_pe * 100
    
    def peg_premium(self, industry_data):
        ipo_peg = self.ipo_data['pe'] / self.ipo_data['growth']
        industry_peg = industry_data['avg_pe'] / industry_data['avg_growth']
        return (ipo_peg - industry_peg) / industry_peg * 100
    
    def pb_premium(self, industry_data):
        ipo_pb = self.ipo_data['pb']
        industry_pb = industry_data['avg_pb']
        return (ipo_pb - industry_pb) / industry_pb * 100
    
    def comparable_premium(self, comparable_companies):
        avg_pe = np.mean([c['pe'] for c in comparable_companies])
        ipo_pe = self.ipo_data['pe']
        return (ipo_pe - avg_pe) / avg_pe * 100
    
    def liquidity_adjustment(self, industry_data):
        liquidity_ratio = self.ipo_data['turnover'] / industry_data['avg_turnover']
        float_adj = self.ipo_data['float_ratio'] / 0.8
        return (liquidity_ratio * float_adj - 1) * 100
    
    def scarcity_premium(self):
        score = 0
        if self.ipo_data.get('has_patent'):
            score += 0.3
        if self.ipo_data.get('market_share', 0) > 0.2:
            score += 0.3
        if self.ipo_data.get('is_domestic_substitute'):
            score += 0.4
        return score * 30
    
    def composite_index(self, industry_data, comparable_companies, market_conditions):
        # 调用上面的 composite_relative_premium_index 函数
        return composite_relative_premium_index(
            self.ipo_data,
            industry_data,
            comparable_companies,
            market_conditions or {'sentiment': 50}
        )

# 使用示例
analyzer = NewStockPremiumAnalyzer(ipo_data)
```

```
┌──────────────────────────────────────────────────────────┐
│ PE 相对溢价率                                            │
│ (新股 PE - 行业 PE) / 行业 PE × 100%                     │
├──────────────────────────────────────────────────────────┤
│ PEG 调整溢价率                                           │
│ (新股 PEG - 行业 PEG) / 行业 PEG × 100%                  │
├──────────────────────────────────────────────────────────┤
│ 可比公司溢价率                                           │
│ (新股 PE - 可比公司平均 PE) / 可比公司平均 PE × 100%     │
├──────────────────────────────────────────────────────────┤
│ DCF 隐含溢价率                                           │
│ (发行价 - DCF 内在价值) / DCF 内在价值 × 100%            │
├──────────────────────────────────────────────────────────┤
│ 流动性调整                                               │
│ (新股换手/行业换手) × (新股流通比/0.8) - 1               │
├──────────────────────────────────────────────────────────┤
│ 稀缺性溢价                                               │
│ 稀缺性得分 × 30% (最高 30%)                              │
├──────────────────────────────────────────────────────────┤
│ 综合指数                                                 │
│ 25%×PE + 25%×PEG + 20%×可比 + 10%×流动性 + 10%×稀缺 + 10%×市场 │
└──────────────────────────────────────────────────────────┘
```

## 八、注意事项

### 1. 数据质量
- 确保行业数据是最新的（TTM 或最新季报）
- 可比公司选择要谨慎（同行业、同规模、同阶段）
- 剔除异常值（亏损、ST、重大重组）

### 2. 板块差异
| 板块 | 合理溢价区间 |
|------|--------------|
| 科创板 | +20% ~ +50% |
| 创业板 | +10% ~ +30% |
| 主板 | -10% ~ +20% |

### 3. 市场周期
- 牛市：溢价率可上浮 10-20%
- 熊市：溢价率应下调 10-20%

### 4. 时间因素
- 上市前 5 日：情绪主导，公式参考意义有限
- 5-20 日：逐步回归理性
- 20-60 日：公式有效性最高
- 60 日后：视为正常股票
