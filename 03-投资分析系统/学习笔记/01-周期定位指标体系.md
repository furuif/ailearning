## 核心指标分类

### 1. 先行指标 (Leading Indicators)
**作用**: 预测经济拐点，通常提前 3-12 个月

**关键指标**:
- **PMI (采购经理指数)**
  - 荣枯线：50%
  - >50% 扩张，<50% 收缩
  - 细分：新订单、生产、就业、供应商配送、库存
  
- **社融增量 (Total Social Financing)**
  - 反映实体经济融资需求
  - 领先 GDP 约 2-3 个季度
  
- **M1-M2 剪刀差**
  - M1 增速 - M2 增速
  - 正值扩大：企业活期存款增加，投资意愿强
  - 负值扩大：资金定期化，经济活力下降
  
- **房地产销售面积同比**
  - 领先房地产投资 6-9 个月
  - 产业链长，影响上下游多个行业

- **消费者信心指数**
  - 反映居民消费意愿
  - 影响未来消费支出

### 2. 同步指标 (Coincident Indicators)
**作用**: 反映当前经济状态

**关键指标**:
- **工业增加值同比**
  - 月度数据
  - 反映工业生产活跃度
  
- **GDP 增速**
  - 季度数据
  - 最全面的经济增长指标
  
- **城镇调查失业率**
  - 反映劳动力市场状况
  - 5.5% 左右为警戒线
  
- **发电量同比**
  - 克强指数成分
  - 难以造假，真实反映经济活动

### 3. 滞后指标 (Lagging Indicators)
**作用**: 确认周期阶段，通常滞后 3-6 个月

**关键指标**:
- **CPI (消费者物价指数)**
  - 通胀指标
  - 滞后于经济复苏
  
- **PPI (生产者物价指数)**
  - 工业品出厂价格
  - 反映企业盈利环境
  
- **企业利润同比**
  - 滞后于需求复苏
  
- **贷款利率**
  - 政策调整后的传导结果

## 周期四阶段特征

### 复苏期 (Early Cycle)
```
先行指标：↑ 回升
同步指标：→ 企稳
滞后指标：↓ 仍弱
特征：政策宽松，预期改善，实际数据尚未好转
配置：股票>信用债>商品>利率债
```

### 过热期 (Mid Cycle)
```
先行指标：↑ 高位
同步指标：↑ 强劲
滞后指标：↑ 开始回升
特征：需求旺盛，产能利用率高，通胀抬头
配置：商品>股票>利率债>信用债
```

### 滞胀期 (Late Cycle)
```
先行指标：↓ 回落
同步指标：→ 见顶
滞后指标：↑ 高企
特征：增长放缓，通胀高企，政策两难
配置：现金>商品>利率债>股票
```

### 衰退期 (Recession)
```
先行指标：↓ 低位
同步指标：↓ 下滑
滞后指标：↓ 开始下降
特征：需求疲软，失业上升，通缩压力
配置：利率债>信用债>现金>股票
```

#### 分位数计算方法
```python
# 滚动历史分位数（以 PMI 为例）
import pandas as pd
import numpy as np

def calculate_percentile(series, window=120):
    """
    计算滚动历史分位数
    series: 时间序列数据
    window: 历史窗口（月数），120=10 年
    """
    def pct_rank(x):
        if len(x) < 12:  # 至少 1 年数据
            return np.nan
        current = x.iloc[-1]
        history = x.iloc[:-1]  # 排除当前值
        rank = (history < current).sum() / len(history)
        return rank
    
    return series.rolling(window).apply(pct_rank)

# 分位数解读
# >80%: 历史高位
# 60-80%: 中高位
# 40-60%: 中枢区域
# 20-40%: 中低位
# <20%: 历史低位
```

### 2. 指标间领先滞后关系量化

#### 互相关分析 (Cross-Correlation)
```python
from statsmodels.tsa.stattools import ccf

def find_lead_lag(series_a, series_b, max_lag=24):
    """
    寻找两个指标间的领先滞后关系
    返回最优领先/滞后月数
    """
    correlations = ccf(series_a, series_b, adjusted=False)
    
    # 找到最大相关性的滞后阶数
    max_corr_idx = np.argmax(np.abs(correlations[:max_lag+1]))
    max_corr = correlations[max_corr_idx]
    
    if max_corr_idx == 0:
        return "同步", max_corr
    elif max_corr_idx > 0:
        return f"A 领先 B {max_corr_idx}个月", max_corr
    else:
        return f"A 滞后 B {abs(max_corr_idx)}个月", max_corr

# 典型领先滞后关系（经验值）
lead_lag_pairs = {

}
```

#### 格兰杰因果检验 (Granger Causality)
```python
from statsmodels.tsa.stattools import grangercausalitytests

def test_granger_causality(data, max_lag=12):
    """
    检验 X 是否格兰杰导致 Y
    data: 两列 DataFrame [Y, X]
    """
    results = grangercausalitytests(data, max_lag, verbose=False)
    
    for lag, result in results.items():
        p_value = result[0]['ssr_chi2test'][1]
        if p_value < 0.05:
            print(f"滞后{lag}期：X 格兰杰导致 Y (p={p_value:.3f})")
```

#### 实战：构建领先指标合成
```python
def composite_leading_indicator(components, weights=None):
    """
    构建合成先行指标
    components: dict {指标名：标准化后的序列}
    weights: 各指标权重（默认等权）
    """
    if weights is None:
        weights = {k: 1/len(components) for k in components}
    
    composite = pd.Series(0, index=next(iter(components.values())).index)
    for name, series in components.items():
        composite += series * weights[name]
    
    return composite
```

### 3. 构建综合周期指数

#### 步骤一：数据标准化
```python
from sklearn.preprocessing import StandardScaler

def standardize_indicators(df):
    """
    将不同量纲的指标标准化为 Z-score
    """
    scaler = StandardScaler()
    standardized = scaler.fit_transform(df)
    return pd.DataFrame(standardized, columns=df.columns, index=df.index)
```

#### 步骤二：确定指标方向
```python
def adjust_direction(df, directions):
    """
    调整指标方向，使所有指标同向（上升=向好）
    directions: dict {指标名：1 或 -1}
    """
    adjusted = df.copy()
    for col, direction in directions.items():
        if direction == -1:
            adjusted[col] = -adjusted[col]
    return adjusted

# 示例：CPI 在过热期是负向指标
directions = {
    'PMI': 1,
    '工业增加值': 1,
    '社融增速': 1,
    'CPI': -1,  # 过高通胀不利
    '失业率': -1,  # 上升是坏事
}
```

#### 步骤三：主成分分析 (PCA) 降维
```python
from sklearn.decomposition import PCA

def build_cycle_index(df, n_components=2):
    """
    使用 PCA 构建综合周期指数
    第一主成分通常代表"增长"
    第二主成分通常代表"通胀"
    """
    pca = PCA(n_components=n_components)
    components = pca.fit_transform(df)
    
    # 第一主成分作为增长指数
    growth_index = components[:, 0]
    # 第二主成分作为通胀指数
    inflation_index = components[:, 1]
    
    return growth_index, inflation_index, pca.explained_variance_ratio_
```

#### 步骤四：四象限定位
```python
def cycle_quadrant(growth, inflation):
    """
    根据增长和通胀指数判断周期位置
    """
    growth_mean = growth.mean()
    inflation_mean = inflation.mean()
    
    quadrants = []
    for g, i in zip(growth, inflation):
        if g > growth_mean and i < inflation_mean:
            quadrants.append("复苏")
        elif g > growth_mean and i > inflation_mean:
            quadrants.append("过热")
        elif g < growth_mean and i > inflation_mean:
            quadrants.append("滞胀")
        else:
            quadrants.append("衰退")
    
    return quadrants
```

#### 完整流程示例
```python
def build_complete_cycle_framework(data):
    """
    完整的周期定位框架
    """
    # 1. 标准化
    standardized = standardize_indicators(data)
    
    # 2. 调整方向
    adjusted = adjust_direction(standardized, directions)
    
    # 3. PCA 分解
    growth, inflation, variance_ratio = build_cycle_index(adjusted)
    
    # 4. 象限定位
    quadrants = cycle_quadrant(growth, inflation)
    
    return {
        'growth_index': growth,
        'inflation_index': inflation,
        'quadrants': quadrants,
        'variance_explained': variance_ratio
    }
```

### 4. 不同周期下的行业轮动规律

#### 周期 - 行业映射表
| 周期阶段 | 受益行业 | 逻辑 | 代表板块 |
|----------|----------|------|----------|
| **复苏** | 金融、可选消费、科技 | 利率低位，需求回暖 | 券商、地产、汽车、电子 |
| **过热** | 周期、资源品 | 需求旺盛，价格上涨 | 有色、煤炭、钢铁、化工 |
| **滞胀** | 必需消费、医药 | 防御属性，需求刚性 | 食品饮料、医药生物 |
| **衰退** | 公用事业、高股息 | 避险需求，稳定现金流 | 电力、电信、银行 |

#### 行业相对强度分析
```python
def calculate_relative_strength(sector_returns, benchmark_returns, window=20):
    """
    计算行业相对强度（RS）
    RS = 行业收益 / 基准收益
    RS 上升 = 行业跑赢大盘
    """
    rs = sector_returns.rolling(window).sum() / benchmark_returns.rolling(window).sum()
    rs_trend = rs.diff()  # RS 的变化趋势
    
    return rs, rs_trend

# 轮动信号：RS 趋势向上 + 周期定位匹配
def rotation_signal(rs_trend, cycle_stage, sector):
    if rs_trend > 0 and sector in sector_rotation_model(cycle_stage)['超配']:
        return "强烈买入"
    elif rs_trend > 0:
        return "买入"
    elif rs_trend < 0 and sector in sector_rotation_model(cycle_stage)['低配']:
        return "强烈卖出"
    else:
        return "持有"
```

```python
def backtest_rotation_strategy(data, initial_capital=1000000):

    """
    # 1. 计算周期定位（每月）

    # 3. 计算下月收益
    # 4. 累计净值
    
    positions = []
    nav = [initial_capital]

        positions.append(weights)
```
