### CDPI 综合指数计算

**第 30 日数据**：
```python
# 输入数据
current_price = 85
issue_price = 50
days_since_ipo = 30
industry_pe = 55
ipo_pe = 75
ipo_growth = 50  # 营收增速 50%
industry_growth = 25  # 行业平均 25%

industry_turnover = 5
volatility = 6  # 日波动率 6%
sentiment_score = 55  # 情绪中性偏多

# 1. PE 相对溢价
pe_premium = (75 - 55) / 55 = 36.4%

# 2. PEG 相对溢价
ipo_peg = 75 / 50 = 1.5
industry_peg = 55 / 25 = 2.2
peg_premium = (1.5 - 2.2) / 2.2 = -31.8%  # 考虑成长后低估

# 3. 可比公司溢价（假设 3 家可比公司平均 PE=65）
comparable_premium = (75 - 65) / 65 = 15.4%

# 4. 流动性调整
liquidity_adj = (12 / 5) * (0.25 / 0.8) = 0.75
liquidity_premium = (0.75 - 1) * 100 = -25%

# 5. 稀缺性溢价（芯片 + 国产替代）
scarcity_score = 0.7  # 技术壁垒 0.3 + 国产替代 0.4
scarcity_premium = 0.7 * 30 = 21%

# 6. 市场情绪调整
market_adj = 1 + (55 - 50) / 100 = 1.05
market_premium = (1.05 - 1) * 100 = 5%

# CDPI 综合计算
cdpi = (
    0.25 * 36.4 +
    0.25 * (-31.8) +
    0.20 * 15.4 +
    0.15 * (-25) +
    0.10 * 21 +
    0.10 * 5
)
cdpi = 9.1 - 7.95 + 3.08 - 3.75 + 2.1 + 0.5 = 3.08%

# 周期调整（假设当前为复苏期，权重 1.2）
adjusted_cdpi = 3.08 * 1.2 = 3.7%
```

### 决策建议

| 指标 | 数值 | 解读 |
|------|------|------|
| 基础溢价率 | 70% | 表面高估 |
| PEG 溢价 | -31.8% | 考虑成长后低估 |
| CDPI | 3.7% | 合理区间 |
| 周期调整 | 复苏期×1.2 | 可承受更高溢价 |
| **综合决策** | **持有** | 高成长支撑当前估值 |

| 第 10 日 | 卖出 20% | 110 元 | 溢价率仍处高位 |
| 第 30 日 | 持有 30% | 85 元 | CDPI 进入合理区间，高成长支撑 |
| 第 60 日 | 观察 | 75 元 | 等待进一步回调或基本面验证 |

### 经验总结

✅ **成功点**：

- 用 PEG 而非 PE 判断估值，避免错失高成长股

⚠️ **改进点**：

## 💡 实战经验总结

### 1. 估值判断的核心逻辑

```
表面溢价率 → PEG 调整 → 可比公司验证 → CDPI 综合
    ↓            ↓            ↓            ↓
 初步判断    成长调整    横向对比    最终决策
```

### 2. 不同板块的溢价率特征

| 板块 | 合理溢价区间 | 关键指标 | 交易策略 |
|------|--------------|----------|----------|
| 科创板 | +20% ~ +50% | PEG、稀缺性 | 高成长 + 高波动 |
| 创业板 | +10% ~ +30% | PEG、行业对比 | 成长 + 估值平衡 |
| 主板 | -10% ~ +20% | PE、PB、股息 | 价值 + 稳健 |

### 3. 周期调整的重要性

| 周期阶段 | 权重 | 策略调整 |
|----------|------|----------|
| 复苏 | 1.2 | 可承受更高溢价，积极申购 |
| 过热 | 0.8 | 溢价率高企，谨慎 |
| 滞胀 | 0.6 | 防御为主，精选低估值 |
| 衰退 | 1.0 | 等待拐点，布局优质 |

### 4. 止盈止损策略

```python
# 止盈策略
if cdpi > 80:
    sell_all()  # 严重高估，清仓
elif cdpi > 60:
    sell_50%()  # 高估，减半
elif cdpi > 40 and profit > 50%:
    sell_30%()  # 合理但盈利丰厚，部分止盈

# 止损策略
if base_premium < -20% and peg_premium > 50%:
    sell_all()  # 破发且高估，止损
elif base_premium < -10% and fundamentals_deteriorate:
    sell_all()  # 破发且基本面恶化
```

### 5. 常见误区

❌ **误区 1**：破发=低估，可以抄底
✅ **正解**：破发可能是发行估值过高，需用 CDPI 判断

❌ **误区 2**：高溢价率=高估，应该卖出
✅ **正解**：高成长可以支撑高溢价，看 PEG 而非 PE

❌ **误区 3**：行业平均 PE 是固定标准
✅ **正解**：行业内部差异大，需选择可比公司

❌ **误区 4**：CDPI 一次计算就够了
✅ **正解**：需动态跟踪，每周更新 CDPI
