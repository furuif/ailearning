## 📊 知识体系

### 模块一：宏观周期定位

```
先行指标（PMI、社融、M1-M2）→ 预测拐点（3-12 个月）
同步指标（工业增加值、GDP）→ 确认现状
滞后指标（CPI、PPI）→ 验证周期（3-6 个月）

周期四阶段：
复苏（增长↑通胀↓）→ 股票
过热（增长↑通胀↑）→ 商品
滞胀（增长↓通胀↑）→ 现金
衰退（增长↓通胀↓）→ 债券
```

### 模块二：新股动态溢价率

```
基础公式：(市价 - 发行价) / 发行价 × 100%

七大进阶公式:
1. 时间调整 → exp(-0.03×天数)
2. 市场调整 (MAP) → (1+ 个股)/(1+ 大盘)-1
3. 行业相对 → 个股溢价/(行业 PE/ 个股 PE)
4. 情绪调整 → 总溢价× (1- 情绪/200)
5. 风险调整 (RAP) → (年化溢价 - 无风险)/ 年化波动
6. 流动性调整 → 换手×流通比调整
7. 综合指数 (CDPI) → 25%PE+25%PEG+20% 可比 +10% 流动 +10% 稀缺 +10% 市场

八大相对估值法:
PE 相对、PEG 调整、PB/ROE、PS/增速、可比公司、DCF 隐含、流动性、稀缺性
```

### 模块三：综合决策

```
调整后 CDPI = CDPI × 周期权重

周期权重:
- 复苏 1.2（可承受更高溢价）
- 过热 0.8（溢价率高企，谨慎）
- 滞胀 0.6（防御为主）
- 衰退 1.0（精选低估值）

决策信号:
>80 严重高估 → 强烈卖出
60-80 高估 → 卖出
40-60 合理 → 持有
20-40 低估 → 买入
<20 严重低估 → 强烈买入
```
